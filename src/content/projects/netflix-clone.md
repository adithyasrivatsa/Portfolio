# Netflix Clone

A simple and elegant Netflix Clone created for educational purposes, using HTML, CSS, and JavaScript. The app dynamically fetches movie data from the [TMDb API](https://www.themoviedb.org/).

## ✨ Features
- **Responsive Layout**: Adapts gracefully to various screen sizes from desktop to mobile devices.
- **Movie Grid**: Displays a grid of movies dynamically populated via the TMDb API.
- **Hero Section**: Contains a call-to-action for a simulated "Get Started" button based on Netflix's landing design.
- **Navigation**: Includes a simple navigation bar with options like Home, TV Shows, Movies, and more.

## 🛠️ Technologies Used
- **HTML5**: For structuring the DOM and semantic layout.
- **CSS3**: For styling the layout, mimicking the iconic Netflix brutalist aesthetic.
- **Vanilla JavaScript**: For interacting asynchronously with the TMDb API to fetch and display accurate movie data.

## 🚀 Setup Instructions

1. **Clone the repository.**
2. **Obtain an API Key**: Register at TMDb to get a developer API key.
3. **Inject Key**: Place your API key within `script.js` where requested.
4. **Run Locally**: Open `index.html` in any modern web browser or use VSCode's Live Server extension.
